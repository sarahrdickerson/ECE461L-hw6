# ECE461L-hw6

### Running the app

1. `cd frontend`
2. `npm install`
3. `npm start`
4. in a new terminal, `cd backend`
5. `cd backend`
6. `. venv/bin/activate`
7. `pip install -r requirements.txt`
8. `flask run --port=8000`
9. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
